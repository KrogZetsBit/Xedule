About This Application

This application allows users to schedule and automatically publish notes to X (formerly Twitter) and Nostr using the official X API. Built with Django and Celery, the platform handles note scheduling, posting, and error management in a fully automated and reliable way.

  • 📅 Note Scheduling: Compose notes and choose when to post them.
  • ⚙️ Automated Publishing: Notes are published automatically at the scheduled time.
  • 🔁 Retry Logic: If posting fails, the system retries with exponential backoff.
  • 📊 Status Tracking: See which notes are pending, published, or failed.
  • 🔒 Secure Authentication: OAuth credentials are used to ensure secure access to the X API.

This tool is ideal for individuals, content creators, and businesses who want to automate their social media workflow and ensure timely and consistent communication on the X and Nostr platforms.
